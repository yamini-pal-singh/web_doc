Media & entertainment

Subtitle generation, interview transcription, multi-language dubbing, content moderation, and post-production search — all powered by Zero STT, Vāķ, and Zero TTS.

Illustrative blueprint — example pipeline patterns
The reference architecture and workflows below are example pipelines showing how Shunya's documented APIs can compose for typical media post-production tasks. They are not Shunya-published media reference designs. The SRT generation snippet uses standard SRT formatting plus Shunya's documented word_timestamps output and is portable to any project.

Common workflows Illustrative

Subtitle generation

  • Upload video or extracted audio to batch ASR with word_timestamps=true.
  • Export directly as SRT or VTT — timestamps from the transcript map cleanly to subtitle cue timing.
  • For multi-language subtitles, run the English transcript through Vāķ for each target language; re-serialize to SRT.

Interview & podcast transcription

  • Batch transcribe with diarization on — clean speaker-tagged transcripts for editorial review.
  • Use enable_summarization for episode summaries.
  • Use enable_keyterm_normalization with your house glossary (brand names, unusual spellings).

Dubbing & voice-over localization

  • Transcribe source audio.
  • Translate script via Vāķ into target language.
  • Either:
    1. Use a preset voice from the 46-voice library that matches the source character, or
    2. Clone the original actor's voice from a 3–6 s reference clip, then speak the translated script in the same voice across 23 languages.
  • Post-mix to the video. For lip-sync, edit translated script length to match source timing (word timestamps help).

Content moderation & scene search

  • Transcribe library with word timestamps.
  • Index transcripts in your search system (Elastic, Typesense, Weaviate for semantic).
  • Queries like "find every time the CEO said 'sustainable growth'" become tractable.
  • For moderation, use enable_profanity_hashing and hash_keywords for banned phrases; flagged clips get human review.

Example — generate SRT from a transcript

from datetime import timedelta

def to_srt_time(s: float) -> str:
    td = timedelta(seconds=s)
    hh, rem = divmod(td.total_seconds(), 3600)
    mm, ss = divmod(rem, 60)
    ms = int((ss - int(ss)) * 1000)
    return f"{int(hh):02d}:{int(mm):02d}:{int(ss):02d},{ms:03d}"

with open("captions.srt", "w") as f:
    for i, seg in enumerate(transcript["segments"], start=1):
        f.write(f"{i}\n{to_srt_time(seg['start'])} --> {to_srt_time(seg['end'])}\n{seg['text']}\n\n")

Rights & ethics

Voice cloning consent
When dubbing with a cloned voice, confirm you have rights — both the original recording's rights and written consent from the voice owner to synthesize. This is both a legal and reputational bar.
