The Shunya dashboard lets anyone — not just developers — upload audio, transcribe it, clean up the output, and export. This page is a picture tour of the main screens.

Illustrative mock — not a screenshot of the real product

The mock screens below are built from HTML/CSS for documentation purposes. They illustrate a plausible operator workflow consistent with Shunya's API capabilities; they are not actual screenshots of playground.shunyalabs.ai, which is sign-in-only.

Real navigation labels, layout, file-size limits, export options, and keyboard shortcuts may differ. We'll replace these mocks with verified screenshots once we have access to the live UI. Do not use this page to estimate dashboard feature parity — sign in and confirm.

4. Export

The Export menu offers every format you're likely to need:

PDF

Formatted with timestamps and speaker labels. Good for compliance and sharing with non-technical readers.

DOCX

Microsoft Word, ready for further editing. Speaker labels become styled paragraphs.

SRT / VTT

Subtitle files for video tools. Uses word timestamps if enabled, otherwise segment timestamps.

TXT

Plain text. Minimal, easy to pipe into other tools.

JSON

Full machine-readable output — every segment, word timestamp, speaker, and intelligence result.

CSV

One row per segment. Useful for analytics pipelines (Looker, Tableau, BigQuery).

5. Speaker management

Under Speakers you can register voice profiles. After registering, any transcription with enable_speaker_identification on will resolve SPEAKER_00 → Priya, SPEAKER_01 → Rahul, etc.

  1. Click + Register speaker.
  2. Upload a 5–15 s clip of the speaker alone. Clean audio, no music, no overlap.
  3. Name the speaker, assign to a project (e.g. support_team), save.

6. API keys

Under API Keys, create and rotate keys. Common pattern: one key per environment, plus a shared read-only key for analytics.

Key rotation hygiene
If a developer leaves or a laptop is lost, revoke the key from this screen immediately. Keys have no grace period — revocation is instant.

Accessibility

  • Keyboard shortcuts: Space to play/pause, J/L to seek, S to add a speaker change, E to edit a word.
  • Dashboard supports dark mode, system auto-detect, and full keyboard navigation.
  • Transcripts are WCAG 2.1 AA contrast-compliant.
