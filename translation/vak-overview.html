Vāķ Translate

Vāķ is Shunya's open-weight translation model for Indian languages. 55 Indian languages, 2,970 any-to-any language pairs, BLEU 38.5 weighted average. Native handling of all major Indic scripts. Released under CC-BY-SA-4.0.

Experimental tier note
The experimental languages have small parallel corpora available. Output is often understandable but rough — suitable for gist translation, not production content. Quality improves as more training data becomes available.

Running Vāķ locally

import ctranslate2
from transformers import NllbTokenizer

tok = NllbTokenizer.from_pretrained("shunyalabs/vak-translate-1.3b-ct2")
mt  = ctranslate2.Translator("shunyalabs/vak-translate-1.3b-ct2", device="cuda")

# Hindi → Tamil
src = tok("आप कैसे हैं?", return_tensors="np")
tokens = tok.convert_ids_to_tokens(src["input_ids"][0])
result = mt.translate_batch([tokens], target_prefix=[["tam_Taml"]])
print(tok.decode(tok.convert_ids_to_tokens(result[0].hypotheses[0][1:])))
# → நீங்கள் எப்படி இருக்கிறீர்கள்?
