Docs/Integrations

Integrations

SDKs, telephony providers, and platforms that work with Shunya Labs out of the box. Every integration on this page is documented in Shunya's published API guide or SDK reference — pick the one that matches your stack.

Python SDK

First-party shunyalabs package on PyPI. AsyncShunyaClient with async/await, retries, and override URLs for self-hosted deployments.

First-party
OpenAI SDK

The TTS API is OpenAI-compatible. Use the official OpenAI Python SDK — point base_url at Shunya and your existing code keeps working.

Drop-in
Twilio

Voice calls and Media Streams. Use Shunya's mulaw 8 kHz output to feed Twilio directly — no resampling step required.

Telephony
SIP / PSTN

Generic telephony pipelines. mulaw for Twilio, North American, and Indian PSTN; alaw for European telephony and many SIP providers.

Telephony
Hugging Face

Open weights at huggingface.co/shunyalabs. Download via huggingface-cli or use the transformers pipeline for local / air-gapped inference.

Self-host

What's not on this page

Integration patterns with named CRMs (Salesforce), CCaaS platforms (Genesys, NICE, Five9, Avaya, Twilio Flex), messaging (WhatsApp Business), or EHRs (Epic, Cerner) are not officially documented by Shunya Labs. We have example architectures for those on the Hands-on evaluation page, marked clearly as Illustrative — they're starting points to validate with your account team, not endorsed integration guides.

Source: every integration above is documented in the Shunya Labs ASR Gateway API Reference and TTS Developer Documentation v1.0 (March 2026), or on the official Hugging Face organization page.
