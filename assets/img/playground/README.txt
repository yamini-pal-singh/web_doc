Playground screenshots
======================

Drop the screenshots from your dashboard chat into this directory using these
exact filenames. The /personas/operator.html walkthrough references them.

  01-empty-state.png        — Speech to Text view, no file uploaded yet
  02-model-dropdown.png     — Model selector open showing Zero Indic / Med / Codeswitch
  03-language-dropdown.png  — Language picker open with search + Most Used / African
  04-uploaded.png           — file uploaded confirmation with progress bar + Replace File
  05-running.png            — Transcribing your file... spinner + Running… button
  06-transcript.png         — Final transcript view with audio player + Detected Language

Recommended size: 1600px wide is plenty. PNG or JPG either works.
